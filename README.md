## 运行方式

### 方式一

直接运行Release文件夹内的24Game.exe

### 方式二

利用Qt官方IDE [Qt Creator](https://www.qt.io/qt-features-libraries-apis-tools-and-ide/#ide) 导入24Game.pro，然后在IDE内运行
